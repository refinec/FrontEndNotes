## 一、$children

### 1. Vue 2.x

> 在 2.x 中，开发者可以使用 `this.$children` 访问当前实例的直接子组件

```vue
<template>
  <div>
    <my-button></my-button>
  </div>
</template>

<script>
import MyButton from './MyButton'

export default {
  components: {
    MyButton
  },
  mounted() {
    console.log(this.$children) // [VueComponent]
  }
}
</script>
```

### Vue 3.x

> 在 3.x 中，使用 **$refs** 访问子组件
