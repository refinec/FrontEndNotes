# 7种请求方法的别名

在使用别名方法时，**url、method、data** 这些属性都不必在配置中指定

1. **axios.request(config)**

2. **axios.get(url[, config])  **： GET的请求方法

   ```javascript
   // 为给定 ID 的user 创建请求
   axios.get('/user?ID=12345').then(function(res){
       console.log(res)
   }).catch(function(err){
       console.log(err)
   })
   // 或者
   axios.get('/user',{
       params:{
           ID:12345
       }
   }).then(function(res){
       console.log(res)
   }).catch(function(err){
       console.log(err)
   })
   ```

   ```javascript
   async function getUser(){
       try{
           const res = await axios.get('/user?ID=123')
       }catch(err){
           console.error(err)
       }
   }
   ```

3. **axios.post(url[, data[, config]])**

   ```javascript
   axios.post('/user',{
       firstName:'f'
   }).then((res)=>{
       ...
   }).catch((err)=>{
       ...
   })
   ```
