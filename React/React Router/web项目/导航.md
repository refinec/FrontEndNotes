## `Link`和`NavLink`

```react
<div>
	<Link to="/">主页</Link>
</div>
```

```react
<div>
	<NavLink 
    exact
    to="/"
    activeClassName={classes.active}
    activeStyle={{color: 'red'}}
   >主页</NavLink>
  <NavLink to="/about">关于</NavLink>
</div>
```

`activeClassName`和`activeStyle`的作用相同，指定链接激活后的样式
